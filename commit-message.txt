feat(dashboard): integrated data injection flow visualization + UI improvements

This commit adds a comprehensive data injection flow visualization directly
integrated into the dashboard ProcessFlow section, along with UI fixes for
ReactFlow node backgrounds and dropdown menu transparency issues.

**Major Feature: DataInjectionFlow Component**

New visualization component (`src/components/process/DataInjectionFlow.tsx`)
that displays the MRV data injection workflow as an interactive flow diagram,
similar to product analytics dashboards showing metric relationships.

Flow Structure:
- **Left Column (Project Activities)**: Data source cards with progress indicators
  - Life Cycle Assessment (LCA) - CO₂e flux analysis
  - Project Emissions - Scope 1, 2, 3 tracking
  - Removal Data - Sensor & lab data collection
  - GHG Statements - Declarations

- **Middle Column (Input Metrics)**: Calculated metrics with expandable details
  - Gross Removal - Sum of all removal activities
  - Total Emissions - Aggregated project emissions
  - Leakage Deduction - 5% leakage factor
  - Buffer Pool - 15% permanence reserve
  - Each node shows values for 7 days, 6 weeks, 12 months with % changes
  - Expandable sections reveal formulas and API/Excel upload options

- **Right Column (Output Metrics)**: Final KPIs with correlation scores
  - Net CORC - Final carbon removal credits (0.999 correlation)
  - Verification Score - 7-category verification status

- **Edge Connections**: Animated dotted lines with correlation values
  - Green edges (≥0.9): Strong positive correlation
  - Amber edges (0.5-0.9): Moderate correlation
  - Red edges (<0): Negative correlation (e.g., emissions deduction)

**Dashboard Integration**

Updated dashboard page (`src/app/(dashboard)/dashboard/page.tsx`):
- Added view toggle between "Data Flow" and "Process Flow" modes
- Data Flow shows DataInjectionFlow (default view)
- Process Flow shows original 8-phase workflow
- Dynamic values based on selected block's tonnage data
- Calculates Net CORC, emissions, leakage, buffer from block data

**UI Fixes**

1. **ReactFlow Node Backgrounds** (`src/app/globals.css`):
   - Fixed white background on ReactFlow nodes
   - Added transparent background rules for:
     - `.react-flow__node` - Base node styling
     - `.react-flow__node-input`, `.react-flow__node-output`, `.react-flow__node-default`
     - `.react-flow__pane`, `.react-flow__viewport`, `.react-flow__renderer`
   - Removed padding, border, box-shadow from default node types

2. **Dropdown Menu Transparency** (`src/app/globals.css`):
   - Added new `.glass-dropdown` class for opaque dropdown backgrounds
   - Background: `rgba(15, 25, 20, 0.95)` - Nearly opaque dark green-black
   - Enhanced shadow for depth: `0 20px 40px rgba(0, 0, 0, 0.5)`
   - Inner glow: `0 0 0 1px rgba(255, 255, 255, 0.1) inset`
   - Prevents visual confusion when dropdowns overlay transparent elements

3. **GlassCard Component Updates**:
   - Added 'dropdown' variant to GlassCard component
   - Updated GlassCardProps type definition
   - Applied dropdown variant to ProjectSelector and RegistrySelector
